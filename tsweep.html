Text Sweeper

This page is dedicated to Text Sweeper, a Minesweeper clone for the TRS-80 Model 100/Tandy 102 Portable Computer. When I noticed there weren't too many games written for the system, I decided to make Text Sweeper. It contains almost all of the features from Microsoft Minesweeper, although due to the size of the display, the minefield is 8x32. The default number of mines is 36, but you can enter a custom amount from 10 to 217.
